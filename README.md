# Qatar Traffic Violations Checker

A premium, responsive landing page and inquiry experience for Qatar traffic violations, built with React, TypeScript, Vite, Mantine UI, and SCSS modules.

## Features

- Arabic and English language toggle
- RTL-aware layout support
- Premium hero, search, features, FAQ, CTA, and footer sections
- Responsive mobile-first design
- Mock violation search flow for demo purposes

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## Project Structure

- `src/components` — reusable UI sections
- `src/pages` — page-level composition
- `src/constants` — translations and content
- `src/styles` — global SCSS setup
- `src/types` — TypeScript interfaces
